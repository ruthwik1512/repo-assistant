# repo-assistant

An AI-powered repository assistant that can clone, traverse, and analyze source code from GitHub repositories.

## Project Structure

```
repo-assistant/
├── repo_assistant/        # Core package
│   └── traversal/         # Module: clone & walk repositories
├── tests/                 # Unit tests
├── main.py                # Entry point / demo runner
└── requirements.txt       # Python dependencies
```

## Setup

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```
